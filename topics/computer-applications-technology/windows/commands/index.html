Windows 命令

快捷进入目录​

  • %UserProfile% → C:\Users\<User>
  • %Appdata% → C:\Users\<User>\AppData
  • %LocalAppdata% → C:\Users\<User>\AppData\Local
  • %Temp% → C:\Users\<User>\AppData\Local\Temp
  • %CommonProgramFiles%/%CommonProgramFiles(x86)%
  • %ProgramData% → C:\ProgramData
  • %ProgramFiles%/%ProgramFiles(x86)% → C:\Program Files/C:\Program Files (x86)
  • %Public% → C:\Users\Public

控制面板 (cpl 文件命令) Control Panel Item​

‘控制面板项’文件 (C:\Windows\System32)

  • 系统属性：sysdm.cpl
  • 添加或删除程序：appwiz.cpl
  • 本地连接：ncpa.cpl / control netconnections
  • 声音和音频设备属性：mmsys.cpl
  • Windows 安全中心：wscui.cpl
  • Windows 防火墙：firewall.cpl
  • 电源选项属性：powercfg.cpl
  • 用户帐户管理：nusrmgr.cpl
  • Internet 属性：inetcpl.cpl
  • 显示属性：desk.cpl
  • 打印机和传真：control printers
  • 任务计划：control schedtasks
  • 管理工具：control admintools
  • 显示属性：control desktop
  • 文件夹选项：control folders

管理控制台 (msc 文件命令) Microsoft Snap-In Control​

‘Microsoft 通用管理文档’文件 (C:\Windows\System32)

  • mmc：Microsoft Management Console
  • 管理控制台：wmimgmt.msc / mmc
  • 计算机管理：compmgmt.msc
  • 设备管理器：devmgmt.msc
  • 磁盘管理：diskmgmt.msc
  • 共享文件夹：fsmgmt.msc
  • 组策略：gpedit.msc
  • 服务：services.msc
  • 系统组件服务：comexp.msc / dcomcnfg
  • 本地用户和组：lusrmgr.msc
  • 证书管理：certmgr.msc
  • 本地安全设置：secpol.msc
  • 组策略结果集：rsop.msc
  • 事件查看器：eventvwr.msc / eventvwr
  • 性能监视器：perfmon.msc / perfmon
  • 高级安全 Windows 防火墙：wf.msc
  • 任务计划程序：taskschd.msc
  • 打印管理：printmanagement.msc
  • 最大内存查询：wmic memphysical get maxcapacity

常用程序(GUI 程序命令)​

‘应用程序’文件 (C:\Windows\System32)

  • 计算器：calc
  • 记事本：notepad
  • 画图板：mspaint
  • 任务管理器：taskmgr
  • 资源管理器：explorer
  • 注册表：regedit
  • 磁盘清理工具：cleanmgr
  • 系统配置实用程序(系统引导、启动)：msconfig
  • 检查 Windows 版本：winver
  • 显示颜色校准：Dccw
  • 查询 DNS 记录、查看域名解析是否正常：nslookup
  • 路由跟踪：tracert
  • 屏幕键盘：osk
  • 问题步骤记录器：psr
  • 资源监视器：Resmon
  • 系统还原：Rstrui
  • 备份状态与配置：sdclt
  • 创建共享文件夹：shrpubw
  • 文件签名验证程序：sigverif
  • 系统激活信息：slui
  • 录音机：soundrecorder
  • 便笺：StikyNot
  • 用户账户控制设置：UserAccountControlSettings
  • 辅助工具管理器：utilman
  • Direct X 诊断工具：dxdiag
  • windows 7 截图程序：snippingtool
  • Windows 更新：wuapp
  • Windows 内存诊断程序：MdSched
  • Windows 系统安全工具：syskey
  • Windows 远程协助：Msra
  • Windows 远程桌面：mstsc
  • 用户帐户控制面板：Netplwiz

软件许可管理工具 (slmgr 文件命名) Software Licensing Management Tool​

C:\Windows\System32\slmgr.vbs

  • slmgr.vbs -dlv ：显示详细的许可证信息
  • slmgr.vbs -dli ：显示许可证信息
  • slmgr.vbs -xpr ：当前许可证截止日期
  • slmgr.vbs -dti ：显示安装 ID 以进行脱机激活
  • slmgr.vbs -ipk ：(Product Key)安装产品密钥
  • slmgr.vbs -ato ：激活 Windows
  • slmgr.vbs -cpky ：从注册表中清除产品密钥(防止泄露引起的攻击)
  • slmgr.vbs -ilc ：(License file)安装许可证
  • slmgr.vbs -upk ：卸载产品密钥
  • slmgr.vbs -skms ：(name[ort] )批量授权

端口占用并结束​

  1. 列出进程极其占用的端口：netstat -ano | findstr <端口号>
  2. 强制关闭进程：taskkill -PID <进程号> -F
